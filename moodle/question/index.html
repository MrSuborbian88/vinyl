Secure iNet Factory

Documentation

The documentation for Secure iNet Factory is split into three sets of documents. These documents are described below.

Name Description
User Guide (CHM) Secure iNet Factory User Guide in compiled HTML format. For Windows platforms only.
User Guide (HTML) Secure iNet Factory User Guide in HTML format.
JavaDoc Generated JavaDoc providing API reference.

Examples

Several console based examples are provided in the examples directory of the Secure iNet Factory distribution. See the README.TXT file for each example for details on it's use.
